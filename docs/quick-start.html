Get Lango Up and Running in Minutes

This quick start guide will help you get Lango installed and running on your local machine in just a few simple steps. Perfect for developers who want to get started quickly!

1
Extract & Navigate

Extract the downloaded ZIP file to your preferred location and navigate to the project directory:

cd lango
2
Install Dependencies

Install all required packages using your preferred package manager:

# Using pnpm (recommended) pnpm install # Or using npm npm install # Or using yarn yarn install
üí° Tip: This project is configured for pnpm. Install it globally with: npm install -g pnpm
3
Set Up Environment Variables

Create a .env.local file in the root directory and add your credentials:

# Database (Get from neon.tech) DATABASE_URL=your_neon_database_url # Clerk Authentication (Get from clerk.com) NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_xxxxx CLERK_SECRET_KEY=sk_test_xxxxx # Admin Access (Get from Clerk Dashboard after first login) ADMIN_USER_ID=your_clerk_user_id # Stripe (Optional - Get from stripe.com) STRIPE_API_SECRET_KEY=sk_test_xxxxx NEXT_PUBLIC_APP_URL=http://localhost:3000

Where to get these values:

  • Database URL: Create a project at neon.tech and copy the connection string
  • Clerk Keys: Create an app at clerk.com and find keys under "API Keys"
  • Admin User ID: Sign in to your app, then find your user ID in the Clerk dashboard
  • Stripe Key: (Optional) Get test keys from stripe.com dashboard
4
Set Up Database

Push the database schema and seed it with sample data:

# Push schema to database npm run db:push # Seed with sample data npm run db:seed

This will create all necessary tables and populate them with 4 language courses (Spanish, Italian, French, Croatian) plus sample lessons and challenges.

5
Start Development Server

Run the development server:

npm run dev

Open your browser and navigate to http://localhost:3000

‚úÖ Success! Your Lango application should now be running. Sign up to create your first account!
6
Access Admin Panel

To access the admin panel:

  1. Sign up/in to your application
  2. Go to Clerk Dashboard ‚Üí Users
  3. Copy your User ID
  4. Add it to ADMIN_USER_ID in .env.local
  5. Restart the dev server
  6. Visit http://localhost:3000/admin
üí° Note: The admin panel lets you create and manage courses, units, lessons, and challenges without touching the database directly.
